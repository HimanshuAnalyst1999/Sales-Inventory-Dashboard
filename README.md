# Sales & Inventory Dashboard (Power BI)

## Project Overview
This project presents an interactive Sales & Inventory dashboard built using Power BI to analyze sales performance, product trends, and inventory levels.

The dashboard helps track key metrics such as total sales, quantity sold, product performance, and stock availability.

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Excel (Dataset)

## Key Features
- KPI cards showing Total Sales, Quantity Sold, Products Sold, and Stock Value
- Sales trend analysis over time
- Top 5 best-selling products
- Category-wise sales analysis
- Sales performance by salesperson
- Supplier-wise quantity sold
- Interactive filters for product category, supplier, and date
